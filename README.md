# ROIIM_ASSIGNMENT

## Clone repositon and follow these steps:-
1. Go to /src/main/resources/application.properties and change DB_name, DB_userName, DB_password as per your mysql database.
2. Run the app.
3. The application will be hosted at localhost:8080
